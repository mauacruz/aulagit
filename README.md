# aulagit
## Este é arquivo Teste para o repositorio de Aula git

#### Comandos utilizados:
- Criei o diretório a ser utilizado -> c:\aulagit;
- Inicializei o repositório -> git init;
- Editei o arquivo README.md;
- Adicionei o arquivo para versionamento -> git add README.md;
- Dei o "commit" -> git commit -m "Criando arquiv README.md - Projeto Fase 1";
- Vinculei o branch atual (master) ao repositorio do GitHum -> git remote add origin https://github.com/mauacruz/aulagit.git
- Subi as alterações do repositorio -> git push -u origin master;
- Fiz mais algumas alterações, fazendo commit e push...
